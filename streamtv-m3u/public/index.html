Steps
  1. Open STB app on Samsung TV
  2. Go to Settings → Playlist → delete old playlist
  3. Add new playlist → choose M3U URL
  4. Paste the URL above → Save
  5. In channel list press Red button → switch view to BY GROUP
  6. Now you'll see: Tamil Live TV, Tamil Movies, News, Sports etc.
⚠️ Some channels will show black screen — free public IPTV streams go offline. Use BY GROUP view and skip broken ones. Add ?validate=1 to URL for pre-filtered list (slower to load).
